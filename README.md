# theme_Template_WordPress_Sass

Un template rapide pour créer un thème WordPress.

## Technologies utilisées
- WordPress
- Sass

## Comment utiliser ce template
1. Téléchargez ou clonez ce référentiel sur votre ordinateur.
2. Ouvrez le dossier avec votre éditeur de code préféré.
3. Utilisez Sass pour développer votre thème.
4. Téléchargez votre thème terminé sur votre site WordPress.
